# Hotel Management System

This is a hotel management System built using Laravel.

## Features
- Display Hotel Rooms.
- Allow visitors to book rooms online.


### Preview
![BookList Demo](public/img/hotel.gif)

### Built with

- Semantic HTML5 markup
- Tailwind CSS
- Laravel Framework version 8
